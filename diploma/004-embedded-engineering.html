Embedded System Foundation Course ဟာ သင့်ကို နေ့စဉ်အသုံးပြုနေရတဲ့ နည်းပညာများရဲ့ အနာဂတ်ကို ကိုယ်တိုင် ဆွဲကိုင်လှုပ်နိုင်သူ တစ်ယောက်ဖြစ်လာဖို့ စတင်မိတ်ဆက်ပေးမယ့် အခြေခံသင်တန်းတစ်ခုပါ။

Embedded System ဆိုတာ သမားရိုးကျ ကွန်ပျူတာတွေနဲ့ မတူဘဲ၊ သီးခြားလုပ်ငန်းတစ်ခု (သို့မဟုတ်) အလုပ်အနည်းငယ်ကိုပဲ တိကျ မြန်ဆန်စွာ လုပ်ဆောင်ဖို့အတွက် ဒီဇိုင်းထုတ်ထားတဲ့ နည်းပညာဦးနှောက်သေးသေးလေးတွေ ဖြစ်ပါတယ်။ ဒီစနစ်လေးတွေကို ကားများ၊ အိမ်သုံးပစ္စည်းများ၊ ကျန်းမာရေးပစ္စည်းများ၊ ဆက်သွယ်ရေးပစ္စည်းများ အပါအဝင် ဘက်ပေါင်းစုံ မှာ အများဆုံးတွေ့ရပါတယ်။

Course Outline

Basic

  1. Module 1: Linux + Embedded (Linux Command, Operating System ၏ အခြေခံများ)
  2. Module 2: Embedded Introduction + Programming Basics + Build Tools (C Programming, Microcontroller/Microcomputer အခြေခံ)
  3. Module 3: Git (Version Control ကို စနစ်တကျ အသုံးပြုခြင်း)
  4. Module 4: C++ / Threading (Object-Oriented Programming, Multi-Threading)

Advanced

  1. Module 5: Embedded Linux (Yocto, Buildroot)
  2. Module 6: Embedded Controller (STM32)

သင်တန်းအကြောင်းအရာအပြည့်အစုံ

Embedded System တွေဟာ သမားရိုးကျ ကွန်ပျူတာတွေထက် ပိုအသုံးဝင်ရတဲ့ အကြောင်းရင်းတွေကတော့ ကျစ်လျစ်သေးငယ်ပြီး စွမ်းအင်ချွေတာနိုင်ခြင်း၊ မြင့်မားသော စွမ်းဆောင်ရည်နှင့် ယုံကြည်စိတ်ချရခြင်း၊ ထုတ်လုပ်မှု စရိတ်သက်သာခြင်း စတာတွေကြောင့် လူကြိုက်များပြီး အသုံးများပါတယ်။ ဒါ့အပြင် ဒီနည်းပညာဟာ လက်ရှိ ခေတ်စားနေတဲ့ IoT (Internet of Things) ရဲ့ အဓိကသော့ချက်ဖြစ်ပါတယ်။ သင့်အိမ်က မီးသီး၊ ရေခဲသေတ္တာ၊ လုံခြုံရေးကင်မရာ အားလုံးကို အင်တာနက်နဲ့ ချိတ်ဆက်ပြီး သင့်ဖုန်းကနေ ထိန်းချုပ်နိုင်တာဟာ ဒီ Embedded System တွေကြောင့်ပါပဲ။ ဒါကြောင့် မဝေးတော့တဲ့ အနာဂတ်မှာ Smart City တွေ၊ ကိုယ်တိုင်မောင်းနှင်တဲ့ ကားများ (Autonomous Vehicles) နဲ့ AI စနစ်ပါဝင်တဲ့ စက်ရုပ်များ (Robotics) အားလုံးဟာ ဒီနည်းပညာအပေါ်မှာပဲ မူတည်ပြီး တိုးတက်လာမှာ ဖြစ်ပါတယ်။

ဒီ Course ဟာ သင့်ကို Embedded Software Engineering ပညာရပ်အတွက် လိုအပ်တဲ့ Skills Set တွေ အပြည့်အစုံ ပေးစွမ်းမှာ ဖြစ်ပါတယ်။ Hardware (MCU/ Board) နဲ့ Software (Linux, C++, Git) ကို တစ်ပြိုင်နက်တည်း ပေါင်းစပ်လေ့လာရတာဖြစ်တဲ့အတွက် သင်တန်းပြီးတာနဲ့-

  • AI, Robotics, IoT နယ်ပယ်တွေမှာ Junior Level နဲ့ စတင်အလုပ်ဝင်ရောက်နိုင်မယ့် Skills တွေ
  • IoT Device ထုတ်လုပ်တဲ့ General Software Development Field တွေမှာ ဝင်ရောက်လုပ်ကိုင်နိုင်မယ့် အခြေခံများ
  • Controller တွေ၊ Micro Computer တွေကို ပြုပြင်ဖန်တီးချင်တဲ့ Service Engineer တွေအတွက် Career Develop ဖြစ်သွားမယ့် နည်းပညာများ
  • စက်ရုံအလုပ်ရုံတွေမှာ Automation စက်တွေအတွက် Firmware ရေးခြင်း၊ IoT စနစ်တွေ တပ်ဆင်ခြင်းများအတွက် လက်တွေ့ကျွမ်းကျင်မှု

သင်တန်းမှာ Arduino ဒါမှမဟုတ် Raspberry Pi လိုမျိုး Platform တွေနဲ့ စမ်းသပ်လေ့လာရင်း သင့်ရဲ့ ကိုယ်ပိုင် Embedded Project လေးတွေကို စတင်ဖန်တီးနိုင်ဖို့အတွက် လမ်းညွှန်ပေးသွားမှာပါ။ နောက်တစ်ကြိမ် သင်ဟာ Smart ကျတဲ့ ပစ္စည်းတစ်ခုကို သုံးလိုက်တဲ့အခါ၊ အဲဒီပစ္စည်းရဲ့ နောက်ကွယ်က နည်းပညာဦးနှောက်သေးသေးလေး ကို သင်ကိုယ်တိုင် ဖန်တီးနိုင်သူတစ်ယောက်ဖြစ်လာဖို့ ဒီသင်တန်းက မရှိမဖြစ်လိုအပ်တဲ့ ပထမဆုံး ခြေလှမ်း ဖြစ်ပါတယ်။
