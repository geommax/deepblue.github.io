Course Outline

Foundation & Filesystems (Week 1 & 2)

  • Deep dive into Linux OS Concepts (Ubuntu, GNOME Desktop)
  • Mastering the Shell and Command Line Structure (cat, tail, head, more, less)
  • Advanced File System Management and User & Group Permissions

Administration & Processes (Week 3 & 4)

  • Expertise in Package Management (apt, yum, dpkg, rpm)
  • Understanding and controlling Linux Processes & Services (top, htop, initd, systemd)
  • Setting up Networking in Linux (nmcli, netplan) and securing access with SSH

Storage & Automation (Week 5)

  • Handling Archiving Files & Scheduling Jobs (regular expressions, grep, cron)
  • Managing Disk Partitions & File Systems (fdisk, cfdisk, gdisk, mount, unmount)

Linux Servers & Security (Week 6 - The Core!)

  • Setting up and configuring key services: NFS, Samba, Vsftp, Http, MARIADB SQL
  • Server Mastery: Deployment with Apache2 and Nginx
  • Firewall Configuration using Firewalld, IPTABLES and UFW

Advanced Topics (Week 7)

  • In-depth look at the Boot Process, recovering root password
  • Learn How to build Your Own Commercial Linux OS
  • Cloud Computing & Containerization & Software Development Eco System

Exam (Week 8)

  • Cloud Computing & Containerization & Software Development Eco System
  • Exam Preparation
  • 2 Hours - Exam
