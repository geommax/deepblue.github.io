Course ID: 022

အနာဂတ်မှာ AI နဲ့ ပတ်သက်ပြီး အခွင့်အလမ်းသစ်တွေကို ရှာဖွေနေသူတွေအတွက် AI Engineering ရဲ့ အဓိက Mathematics Concepts များဖြစ်တဲ့ Algebra, Calculus တို့ကို အခြေခံကနေစပြီးသင်ကြားပေးမယ့်အပြင် နောက်ကွယ်မှာ အလုပ်လုပ်နေတဲ့ Algorithm တွေပါ တစ်ခါတည်း လေ့လာသင်ယူခွင့်ရမယ့် AI Crash Course အတန်းပဲဖြစ်ပါတယ်။

Course Outline

1st Month

  • Supervised vs. Unsupervised , Reinforcement
  • Image Processing & Computer Vision
  • Edge Detection & Kernel Matrix processing
  • Clustering/PCA for dimensionality reduction
  • K-nearest Neighbors Algorithms, Support Vector Machine & Naive Bay Algorithms
  • Decision Tree and Random Forest Algorithm
  • Linear Algebra (Vectors and Matrix, transformation Matrices)
  • Differential Calculus, Gradient Descent, Chain Rule

2nd Month

  • Linear, Logistics Regression & Loss
  • Continuous prediction with linear regression algorithm
  • Perceptron Algorithm (Straight line formula for linear classifiers)
  • Neural Network, Multi-Layer Perceptron & Deep Learning Algorithm
  • Optimization Calculus, Statistical Evaluation Metrices
  • Data Training & Model Testing
  • Convolutional layers, pooling layers, activation layers
  • Object Detection in realtime, Bounding Box Regression

3rd Month

  • Recurrent Neural Network
  • Time series data forecasting
  • RNNs, LSTMs, and the transition into Attention Mechanisms
  • LLMs Model Basic, Transformer architecture
  • Architecture of BERT, GPT, and Llama. Understanding Tokenization and Context Windows
  • Encoder , Decoder, Attention Heads
  • RAG & Vector Databases
  • Parameter-Efficient Fine-Tuning
  • Reinforcement Learning & Tuning LLMs

Projects

  • DT Classification
  • Neural Network
  • CNN From Scratch
  • Transfer Learning - Google Efficient Net
  • PyTorch & Tensor Flow
  • Tuning LLMs
  • AI powered chatbots

3 Months
