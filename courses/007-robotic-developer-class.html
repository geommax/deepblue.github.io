Robotics Developer Class

Course ID: 007

၄ လတာ Robotics Developer Class ဟာ ခေတ်မီစက်ရုပ်တည်ဆောက်ရေး (Robotics Development) အတွက် လိုအပ်တဲ့ နည်းပညာစံနှုန်းတွေကို အခြေခံကစပြီး အဆင့်မြင့်အထိ ပိုင်နိုင်အောင် သင်ကြားပေးမှာပါ။ သင်တန်းရဲ့ ပထမဆုံးအပိုင်းအနေနဲ့ Module 1 (Linux Foundation) မှာ စက်ရုပ်စနစ်တွေရဲ့ အခြေခံ Operating System ဖြစ်တဲ့ Linux ကို လက်တွေ့ကျကျ ကိုင်တွယ်သုံးစွဲပုံနဲ့ မရှိမဖြစ် Command တွေကို ကျွမ်းကျင်အောင် လေ့လာရပါမယ်။ ပြီးရင် Module 2 (C++ Programming) မှာ စွမ်းဆောင်ရည်မြင့်မားတဲ့ စက်ရုပ်ထိန်းချုပ်မှု ကုဒ်တွေရေးဖို့ အဓိကဘာသာရပ်ဖြစ်တဲ့ C++ ရဲ့ အခြေခံသဘောတရားတွေ၊ Data Structure တွေနဲ့ Object-Oriented Programming (OOP) ပုံစံတွေကို နက်နက်နဲနဲ သင်ကြားပေးမှာပါ။ ဒါတွေအားလုံးက အခြေခံကနေစသင်တာဆိုတော့ စိတ်ပူစရာမလိုပါဘူးနော်။

သင်တန်းအကြောင်းအရာအပြည့်အစုံ

သင်တန်းရဲ့ ဒုတိယပိုင်းမှာတော့ စက်ရုပ်တည်ဆောက်ရေးအတွက် စံနှုန်းဖြစ်လာတဲ့ Robot Operating System 2 (ROS2) ကို အဓိကထားပြီး သင်ပေးမှာပါ။ Module 3 (Introduction to ROS2) မှာ ROS2 ရဲ့ အခြေခံလုပ်ဆောင်ပုံတွေ၊ Nodes, Topics, Services, Actions လိုမျိုး Communication Mechanism တွေကို နားလည်အောင် သင်ကြားပေးပြီး၊ Module 4 (ROS2 Intermediate - Simulation) မှာ သင်ယူထားတဲ့ အသိပညာတွေကို လက်တွေ့အသုံးချရမှာ ဖြစ်ပါတယ်။ ဒီ Module မှာ ROS2 ကိုသုံးပြီး စက်ရုပ်အမျိုးမျိုးရဲ့ Simulation (အတုယူ ပတ်ဝန်းကျင်) တွေ ဖန်တီးကာ စက်ရုပ်ကို ထိန်းချုပ်တာ၊ လမ်းကြောင်းရှာတာနဲ့ Sensor အချက်အလက်တွေကို ပရိုဂရမ်နဲ့ ကိုင်တွယ်တာတွေကို လက်တွေ့ကျကျ လေ့ကျင့်ရမှာပါ။ Simulation ရဲ့ အားသာချက်ကတော့ စျေးကြီးတဲ့ စက်ရုပ်အစစ်တွေမလိုဘဲ အမှားအယွင်းတွေကို လုံခြုံစွာစမ်းသပ်ပြီး လေ့ကျင့်နိုင်တာပါပဲ။ သီအိုရီ (စာတွေ့) နဲ့ လက်တွေ့လုပ်ဆောင်မှု (Hands-on) ကို ပေါင်းစပ်ပြီး ၄ လ အတွင်း အပြည့်အဝ ကျွမ်းကျင်စေဖို့ ရည်ရွယ်ပါတယ်။

ဘယ်သူတွေ လေ့လာသင့်လဲ

Robotics Developer Class က စက်ရုပ်တည်ဆောက်ရေးနယ်ပယ်မှာ အလုပ်လုပ်ချင်သူတိုင်း၊ နည်းပညာနယ်ပယ်ကို ပြောင်းချင်သူတိုင်းအတွက် အရမ်းသင့်တော်ပါတယ်။ Programming အခြေခံ မရှိသေးတဲ့သူတွေ (ဒါမှမဟုတ်) Linux ကို တစ်ခါမှ မသုံးဖူးသေးတဲ့သူတွေ ဆိုရင်လည်း အခြေခံကစပြီး သင်ပေးမှာမို့ စိတ်ပူစရာမလိုဘဲ တက်ရောက်လို့ရပါတယ်။ အထူးသဖြင့် Software Developer တွေ ဟာ သူတို့ရဲ့ C++ Programming ကျွမ်းကျင်မှုကို Robotics ရဲ့ အဆင့်မြင့်စနစ်တွေ ထိန်းချုပ်ရေးဘက်ကို ပြောင်းပြီး အသုံးချချင်တယ်ဆိုရင်၊ Mechanical/Electrical Engineer တွေ က စက်ရုပ်ရဲ့ ဒီဇိုင်းအပြင် Software ထိန်းချုပ်မှုအပိုင်းပါ တတ်မြောက်ချင်တယ်ဆိုရင်၊ ဒါမှမဟုတ် နည်းပညာအသစ်တွေကို စိတ်အားထက်သန်စွာ လေ့လာချင်တဲ့သူတွေ ဟာ ကမ္ဘာ့အဆင့်မီ ROS2 နည်းပညာနဲ့ ကိုယ်ပိုင်တီထွင်ဖန်တီးမှုတွေ လုပ်ချင်တယ်ဆိုရင် ဒီသင်တန်းဟာ မရှိမဖြစ် လေ့လာသင့်တဲ့ သင်တန်းတစ်ခု ဖြစ်ပါလိမ့်မယ်။
